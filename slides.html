ASGORITHM TECHNOLOGIES17
Code Pattern

How to create a tool

Use @tool for Python functions and ToolNode to execute model-requested tools.

from langchain_core.tools import tool from langgraph.prebuilt import ToolNode @tool def calculator(expression: str) -> str: return str(eval(expression))

Persistence

Use checkpointers and thread_id to remember conversation state across turns.
